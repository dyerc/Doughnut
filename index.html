Doughnut

Uncluttered podcast listening app for Mac.

GitHubGPLv3Downloadv2.0.1

Requires macOS 10.15+, the latest macOS recommended

Built from scratch for macOS

Doughnut is built to be a modern macOS application adopting Apple’s design language, whilst preserving it’s own unique user experience. 100% open source Swift based.

  • Fast and Intuitive, Native Interface
  • Favourite Episodes
  • Search
  • Custom Podcasts(without an RSS feed of their own)
  • Flexible Library(use a network location or even keep multiple libraries on your machine)
